# project作成

---

`composer create-project symfony/skeleton:"6.1.*" my_project_directory`

## symfony 起動
symfony server:start -d  
or  
`php -S 127.0.0.1:8000 -t public`  
## symfony server log出力(色付き)
symfony server:log  

## symfony 環境変数出力
symfony var:export

## symfony エンティティからmigration作成
symfony console make:migration
